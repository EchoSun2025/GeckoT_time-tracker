# 🦎 GeckoT Time Tracker

一个完全离线的时间记录桌面应用，帮助你追踪和统计个人时间分配。

![Platform](https://img.shields.io/badge/Platform-Windows%20%7C%20Linux%20%7C%20macOS-blue)
![License](https://img.shields.io/badge/License-MIT-green)

## ✨ 功能特性

### 🎯 核心功能
- **快捷记录** - 全局快捷键一键开始/停止计时
- **标签分类** - 自定义标签对时间进行分类（work、learn、coding 等）
- **特殊标签** - 休闲/娱乐类标签，不计入总时间统计
- **智能提醒** - 自动提醒休息或督促回归工作
- **数据报表** - 日报、周报、月报多维度统计

### 📊 报表功能
| 报表类型 | 功能说明 |
|---------|---------|
| 日报 | 24小时时间轴可视化（AM/PM双栏）、时间线列表、标签统计 |
| 周报 | 7天柱状图、日均时长、标签分布 |
| 月报 | 日历热力图、月度汇总、趋势分析 |

### 🎮 迷你模式
- 桌面小组件，始终置顶显示
- 实时显示当前计时状态
- 一键继续上一个任务
- 拖拽移动位置

### 🔔 智能提醒系统
| 标签类型 | 默认间隔 | 提醒内容 |
|---------|---------|---------|
| 🎮 特殊标签（休闲） | 每 30 分钟（可自定义） | 鼓励回归工作的骚话，语气逐次加重 |
| 📊 普通标签（工作） | 每 1.5 小时（可自定义） | 提醒休息、喝水、活动颈椎 |

**自定义设置（设置 → 提醒设置）：**
- ⏱️ 自定义提醒间隔（分钟）
- 📝 选择「随机骚话」或「自定义内容」
- 🎯 普通标签和特殊标签分别设置

### 💾 数据管理
- **完全离线** - 数据存储在本地，无需联网
- **导入导出** - JSON 格式，跨设备迁移
- **智能合并** - 导入时自动合并相同标签

## ⌨️ 快捷键

### 全局快捷键（任何界面都可用）
| 快捷键 | 功能 |
|-------|------|
| `Ctrl + Shift + T` | 开始新的时间记录 |
| `Ctrl + Shift + S` | 停止当前计时 |
| `Ctrl + Shift + M` | 切换迷你模式 |

### 新建记录对话框
| 快捷键 | 功能 |
|-------|------|
| `↑` `↓` | 在描述输入框和标签选择区之间切换 |
| `←` `→` | 在标签之间移动选择 |
| `Space` | 选中/取消选中当前标签 |
| `N` | 快速新建标签 |
| `Enter` | 开始计时 |
| `Esc` | 取消并关闭对话框 |

### 时间轴操作
| 快捷键 | 功能 |
|-------|------|
| `Alt + 滚轮` | 缩放时间轴（0.5x - 5x） |

## 🚀 快速开始

### 方式一：直接使用（推荐）
1. 下载 `BUILD/win-unpacked/` 文件夹
2. 双击 `Time Tracker.exe` 运行

### 方式二：从源码构建
```bash
# 1. 安装依赖
npm install

# 2. 开发模式运行
npm start

# 3. 打包成 EXE
npm run build
```

或者使用批处理脚本：
- `1_安装依赖.bat` - 安装 npm 依赖
- `2_打包EXE.bat` - 打包成 Windows 可执行文件
- `3_启动应用.bat` - 开发模式启动
